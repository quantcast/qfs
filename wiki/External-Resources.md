# QFS External Resources

This page provides external links to the interesting work/resources that the
community put together using QFS. We believe that these links can be useful to
everybody who want to get on-hands experience with QFS and learn more about it.
We thank the contributors of this list. Please create a pull request if you'd
like to see a link to your work as well.

* [QFS on the ODROID XU4 Cluster](http://diybigdata.net/2016/07/quantcast-file-system-and-spark-on-odroid-cluster/)
* [Notes on QFS design in Chinese](http://blog.csdn.net/lpstudy/article/details/51457250)
